# WaveBuddy

**Audio & Video Editor for macOS** | v2.0

[Website](https://wave-buddy.com) | [Support](https://wave-buddy.com/support.html) | [Privacy Policy](https://wave-buddy.com/privacy.html)

---

WaveBuddy is a powerful native macOS audio and video editor with multi-track timeline, live recording, speech-to-text transcription, and stunning waveform visualization.

## Features

### Recording
- **Live Audio Recording** - Record from microphone, system audio, or screen with real-time waveform preview
- **Screen Recording** - Capture full screen or select a region using ScreenCaptureKit
- **Live Waveform Preview** - See audio levels in real-time while recording

### Multi-Track Timeline
- **Multi-Track Editing** - Arrange multiple audio and video clips on separate tracks
- **Video Thumbnail Strips** - Video clips display thumbnails with scrub zoom effect
- **Clip Snapping** - Yellow guide lines show snap targets for precise alignment
- **Timeline Zoom** - Pinch to zoom, scroll wheel, or keyboard shortcuts
- **Keyboard Clip Control** - Arrow keys to move clips, ⌘C/⌘V to copy/paste
- **Ghost Drag Preview** - Smooth clip dragging with ghost outline

### Waveform Editor
- **Waveform Visualization** - Filled or line styles with 12+ color themes
- **Edited Region Overlays** - Visual tracking of all edits with colored overlays
- **Selection Actions** - Floating toolbar with Copy, Cut, Delete, Copy to New Window
- **Precise Trimming** - Trim handles or enter exact timestamps

### Audio Effects
- **10-Band Equalizer** - Full parametric EQ with presets (Voice, Podcast, Loudness, Rock, Jazz, Classical, Electronic)
- **8 Voice Effects** - Robot, Chipmunk, Deep Voice, Echo, Radio, Underwater, Megaphone, Whisper
- **Normalize & Amplify** - Automatic loudness normalization or manual gain adjustment

### Speech & Video
- **Speech to Text** - On-device AI transcription using Apple's Speech framework (no internet required)
- **Unified Video Player** - Single preview for both timeline and editor with scrub preview
- **Multi-Track Video Export** - Export master mix as MP4/MOV with black frames for gaps

### Non-Destructive Workflow
- **Export (Safe)** - Creates new files, never modifies originals
- **Recoverable Projects** - Embed project data in exports for full recovery later
- **Auto Recovery** - Drag exported file back to restore complete project

### UI & Themes
- **12+ Color Themes** - Classic, Ocean, Sunset, Forest, Neon, Midnight, Cloud, Glass, Sakura, Orchid, Crimson
- **Background Transparency** - Adjustable UI transparency from 30% to 100%
- **Native macOS** - Built with SwiftUI for fast, responsive experience

## Supported Formats

**Import:** MP3, WAV, FLAC, M4A, AAC, AIFF, MP4, MOV

**Export:** WAV, FLAC, M4A, MP4, MOV

## System Requirements

- macOS 14.0 (Sonoma) or later
- Apple Silicon (M1/M2/M3) or Intel Mac
- 4GB RAM minimum (8GB recommended)

## Privacy

WaveBuddy does not collect any personal data. All files are processed locally on your device. Speech-to-text uses Apple's on-device AI with no data sent to external servers.

See our full [Privacy Policy](https://wave-buddy.com/privacy.html).
