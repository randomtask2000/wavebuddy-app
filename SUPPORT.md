# WaveBuddy Support

Need help with WaveBuddy? You're in the right place.

## How to Get Support

### Report a Bug
1. Go to [Issues](https://github.com/randomtask2000/wavebuddy-app/issues)
2. Click "New Issue"
3. Include:
   - Description of the problem
   - Steps to reproduce
   - Your macOS version
   - File type you were working with

### Request a Feature
Have an idea? [Open a feature request](https://github.com/randomtask2000/wavebuddy-app/issues/new) and tell us what you'd like to see.

### Common Questions

**How do I trim audio/video?**
Use the trim handles on the waveform or enter specific start/end times in the trim panel.

**What formats can I export?**
- Audio: WAV, FLAC, M4A
- Video: MP4

**Why can't I hear audio?**
Make sure your system volume is up and the correct output device is selected in System Settings.

**How do I change the waveform color?**
Use the color picker in the toolbar to customize your waveform appearance.

## System Requirements

- macOS 14.0 (Sonoma) or later
- Apple Silicon or Intel Mac
