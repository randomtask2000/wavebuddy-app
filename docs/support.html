How to Report a Bug

When reporting a bug, please include:

  • A description of the problem
  • Steps to reproduce the issue
  • Your macOS version (e.g., Sonoma 14.2)
  • WaveBuddy version (from About WaveBuddy)
  • The file type you were working with (if applicable)
  • Any error messages you see
cat faq.md
Frequently Asked Questions

What audio formats can I import?

MP3, WAV, FLAC, M4A, AAC, and AIFF.

What video formats can I import?

MP4, MOV, and other QuickTime-compatible formats.

What formats can I export to?

Audio: WAV, FLAC, M4A. Video: MP4, MOV.

Why can't I hear audio?

Check system volume and output device in System Settings > Sound.

How do I change the waveform color?

Settings > Themes. Choose from 12+ color themes or customize.

How do I record audio?

Click the record button in the toolbar. Select input: microphone, system audio, or screen.

Can I recover a project from an exported file?

Yes! Export with "Recoverable" option. Drag the file back into WaveBuddy to restore.

Does transcription require internet?

No. Speech-to-text uses Apple's on-device AI. Completely offline and private.
